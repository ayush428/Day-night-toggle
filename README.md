


Day Night Toggle Website
This project is a simple website that features a day and night toggle functionality implemented using HTML, CSS, and JavaScript. Users can switch between light and dark modes with a single click, providing them with a personalized browsing experience.

Features
Day/Night Toggle: Users can easily switch between light and dark modes by clicking on the toggle button.
Persistent Theme Preference: The selected theme preference is stored in the browser's local storage, ensuring that the chosen theme persists even after page reloads.
Responsive Design: The website is designed to be responsive and work seamlessly across various devices and screen sizes.
Technologies Used
HTML: Used for structuring the content of the website.
CSS: Used for styling the website and creating visual effects.
JavaScript: Used for implementing the toggle functionality and managing theme preferences.
Usage
Clone the repository to your local machine.
Open the index.html file in your web browser.
Click on the toggle button to switch between light and dark modes.
